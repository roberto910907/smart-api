Generate a Laravel API in minutes

## Installation

You can install the package via composer:

```bash
composer require softdevel/smart-api
```

You can publish the config file with:

```bash
php artisan vendor:publish --tag="smart-api-config"
```

## Testing

```bash
composer test
```
